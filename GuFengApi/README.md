# ComicSpider
一个自动爬取漫画图片的工具。  
漫画资源源自 [古风漫画网](https://www.123gf.com/)  
欢迎 Pull Request  

[点此下载最新版](https://github.com/Hidden-Corner/Comic-Spider/releases/latest/download/Comic-Spider.zip)

## 运行环境

.Net Framework 4.7.2

## 原理

通过静态 HTML 分析，模拟浏览器行为。
